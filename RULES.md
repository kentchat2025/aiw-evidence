# AIW Evidence Repo Rules (PUBLIC)

Allowed:
- Sanitized snapshots of UI/Backend source (no secrets)
- Deployed proof (hashes + listings only)
- Execution summaries (counts/stats; redacted rows)
- OCI manifests (object name + etag + time; NO PAR URLs)

NEVER allowed:
- .env / tokens / keys / PAR URLs / broker creds
- Any customer PII
